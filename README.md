# Madison Exterior Co.

A responsive multi-page business website built with HTML, CSS, and vanilla JavaScript.

**Live Demo:** https://matgeroina.github.io/madison-exterior/

## Overview

This project is a static company website designed to showcase exterior remodeling services. It focuses on clean structure, modular layout, and responsive design without using modern front-end frameworks.

## Features

- Modular header and footer using JavaScript includes  
- Responsive layout powered by Bootstrap  
- Service sections with FAQ accordions  
- Gallery with modal preview  
- Mobile-friendly and accessible layout  

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Bootstrap 5  
- GitHub Pages  

## Project Structure
/partials shared layout components
/assets styles and scripts
/*.html static pages

## Purpose

Built as a portfolio project to demonstrate front-end fundamentals, including semantic HTML, responsive design, and basic DOM manipulation.
